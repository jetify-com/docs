## Synopsis[​](#synopsis "Direct link to Synopsis")

Download environment variables stored into the specified file (most commonly a .env file). The
format of the file is one NAME=VALUE per line.

```bash
devbox secrets download <file1> [flags]
```

## Options[​](#options "Direct link to Options")

```
      --environment string   Environment name, such as dev or prod (default "dev")  -f, --format string        File format: env or json (default "env")  -h, --help                 help for download      --org-id string        Organization id to namespace secrets by      --project-id string    Project id to namespace secrets by
```
